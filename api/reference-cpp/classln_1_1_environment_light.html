Detailed Description

環境ライトのオブジェクトです。

環境ライトは、アンビエントライティング、半球ライティング、ディレクショナルライトが統合された、 シーン全体のライティングをコントロールするためのオブジェクトです。

World はメインライトとして、1 つの EnvironmentLight がデフォルトで存在しています。
