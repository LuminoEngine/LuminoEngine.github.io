Detailed Description

template<typename T>
class ln::Optional< T >

任意の型Tの値と、追加の無効値状態を表現できる型です。

次の例では、値を表示する前に変数に値が格納されているかどうかをテストします。

Optional<int> x = 10;
if (x)
std::cout << x.value();
else
std::cout << "Undefined";
