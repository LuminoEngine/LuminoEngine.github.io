490 // Note: C や HSP などラムダ使えない Binding ではキャプチャによって任意のデータを関連付けることができない。

493 // 従来、C++ での利便性のために引数無しハンドラを作っていたが、それだとカバーできない。
494 // 一応↓のように auto 使うことでこのくらいまで簡略化はできるので、これでひとまず進めてみる。
495 // element->connectOnSubmit([](auto x) { ... });
