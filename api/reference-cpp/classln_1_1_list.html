Detailed Description

template<typename T>
class ln::List< T >

参照カウントを持つ可変長配列のコンテナテンプレートクラスです。

インデックスに基づく高速アクセスとインデックスベースの挿入と削除などの機能を提供します。 std::vector とおなじように使用できますが、ヒープに確保された List のインスタンスはベースクラスの RefObject が持つ参照カウントによっても寿命を管理されます。
