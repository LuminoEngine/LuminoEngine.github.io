Detailed Description

特定のエンコーディングのストリームを文字列として読み込む TextReader です。

エンコーディングを明示しない場合、ファイルは UTF-8 テキストとして読み込まれます。 ただし明示した場合でも、UTF 系の BOM を検出した場合は BOM が示すエンコーディングが優先されます。
