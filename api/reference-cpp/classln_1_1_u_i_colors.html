Detailed Description

定義済みの色を取得するためのクラスです。

カラーパレットは MaterialDesign の色で定義されています。 https://material.io/design/color/the-color-system.html

MaterialDesign をベースとしていますが、連続的なアクセスを簡単にするため カラーインデックス (引数 shades) は 0~9 で指定する点に注意してください。
